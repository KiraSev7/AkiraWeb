Belajar Bahasa Jepang (Bagian 4)

Kasual atau formal?

Ketika menghubungkan dua kalimat dengan から, kita dapat menggunakan bentuk kasual pada bagian pertama dan bentuk formal pada bagian kedua. Ini terdengar natural untuk penutur asli, dan masih dianggap sebagai kalimat formal!

  • 今日はゲームをするから、さんぽしたくないです。 (Saya tidak ingin pergi berjalan-jalan karena saya akan main gim video hari ini.)
  • けがをしたから、びょういんに行きます。 (Saya akan pergi ke rumah sakit karena saya mencederai diri sendiri.)
  • 日本語でアニメが見たいから、日本語の勉強をします。 (Saya akan belajar bahasa Jepang karena saya ingin menonton anime dalam bahasa Jepang.)

けいようし (kata sifat)

Sekarang kita akan belajar cara menggunakan から dengan kata sifat. Untuk kata sifat i, kita hanya menambahkan から setelah bentuk kasual.

  • このくつは大きいから、ほしくないです。 (Karena sepatu ini besar, saya tidak menginginkannya.)
  • このかばんは新しいから、好きです。 (Karena tas ini baru, saya menyukainya.)

Kiat untuk Anda!

Kita juga bisa menggunakan から dengan kata kerja berakhiran ます! Strukturnya mudah: bentuk kata kerja ます + から

  • しごとがありますから、行けません。 (Saya tidak bisa pergi karena saya ada pekerjaan.)
  • けがをしましたから、びょういんに行きます。 (Saya akan pergi ke rumah sakit karena saya mencederai diri sendiri.)

Kiat untuk Anda!

Untuk menggunakan から dengan kata sifat na, kita gunakan struktur ini: kata sifat na +だ+から、

  • ふじ山はきれいだから、しゃしんをとりたいです。 (Karena gunung Fuji cantik, saya ingin memotretnya.)
  • この本が好きだから、ほしいです。 (Karena saya menyukai buku ini, saya menginginkannya.)

みじかいぶんしょうです (Ini kalimat pendek)

Sejauh ini kita sudah melihat bagaimana mengatakan alasan untuk sesuatu yang terjadi, seperti すしがすきだから、日本に行きたいです。 (Karena saya suka sushi, saya ingin pergi ke Jepang.) Jika kita hanya ingin mengatakan alasan untuk sesuatu (Karena saya suka sushi.), kita bisa menggunakan pola ini: bentuk kasual kata kerja atau kata sifat から + です。dan katakan: すしがすきだからです。

  • 好きだからです。 (Karena saya menyukainya.)
  • いそがしかったからです。 (Karena saya sibuk.)
  • ほしいからです。 (Karena saya menginginkannya.)

がんばりましたね!

Sekarang Anda tahu beragam cara untuk memberi alasan!

  • 本がほしいですから、かい ます。 Menggabungkan dua kalimat
  • 本がほしいから、かいま す。 Menggabungkan bentuk kasual dan formal
  • あの本はおもしろいか ら、ほしいです。 kata sifat-i
  • あの本はきれいだから、 ほしいです。 kata sifat-na
  • 新しいからです。 jawaban singkat
